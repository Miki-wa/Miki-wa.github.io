# Miki-wa.github.io
ポートフォリオサイトです。

## URL
https://Miki-wa.github.io/

## 概要
HTML/CSSを用いて制作したポートフォリオサイトです。  
専門スクールで学んだ内容をもとに、ユーザーが迷わず情報にアクセスできるよう、シンプルで分かりやすい構成を意識して制作しました。

## 使用技術
- HTML
- CSS
- JavaScript

## 工夫した点
- レスポンシブ対応（スマートフォンでも見やすく表示できるよう調整）
- 情報のまとまりや余白を意識し、視認性を高めたレイアウト設計
- 初めて閲覧するユーザーでも分かりやすいシンプルな構成

## 今後の目標
JavaScriptを用いた動的な表現への理解を深め、ユーザーの使いやすさを意識したサイト制作ができるよう取り組んでいきたいです。  
また、Node.js/Expressにも触れた経験があり、今後はサイトの更新や運用に活かせるよう理解を深めていきたいです。  
将来的には、より良いWeb体験を提供できるようになりたいと考えています。
